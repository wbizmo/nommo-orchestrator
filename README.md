# NOMMO Orchestrator

**NOMMO — Network-Oriented Management & Monitoring Orchestrator**

Distributed service orchestration for resilient systems.

NOMMO is a TypeScript distributed systems control plane focused on service discovery, node health monitoring, failure detection, routing decisions, and self-healing orchestration experiments.
